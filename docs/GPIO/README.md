# [Raspberry's GPIO configuration for HomeAutomation] [0]


## Quick start

Install the GPIO module:

		sudo apt-get install rpi.gpio

To enable/didsble certain interfaces,

		sudo raspi-config

and go to "5 Interfacing Options"
